=========================
===== Tugas Project =====
=========================

1. Setiap kelompok diminta untuk mengembangkan sebuah aplikasi/program berbasis tema bebas (misalnya: sistem manajemen perpustakaan, aplikasi pemesanan, game sederhana, sistem rekomendasi, dll).

2. Aplikasi wajib mengimplementasikan konsep-konsep berikut:
- Pengelolaan data menggunakan array, linked list dan binary search tree
- Pencarian data menggunakan linear dan binary search
- Pengurutan data menggunakan Insertion Sort dan Merge Sort
- Relasi antar data menggunakan graph, boleh adjacency matrix boleh adjacency list (Contoh: jaringan, peta rute, relasi antar objek)

3. Kelompok dibentuk sendiri sendiri dengan ketentuan setiap kelompok beranggotakan 4-5 orang termasuk ketua. Luaran yang dikumpulkan :
- Source code project atau link GitHub
- Slide Presentasi

4. Pengumpulan hasil akhir project :
- Deadline hari Jumat di minggu "UAS"
- Luaran diupload di https://drive.google.com/drive/folders/1Rm_KwRUMCjs2slvOoEanUyEnhhZNhTvP?usp=drive_link

5. Komponen yang dinilai :
- Kelengkapan implementasi
- Ketepatan implementasi
- Kontribusi individu masing-masing anggota
- Frekuensi konsultasi / responsi tugas project (dihitung kehadiran per individu)
- Nilai tambah apabila memanfaatkan platform project kolaboratif seperti GitHub dan kontribusi aktif anggota terlihat di platform tersebut
